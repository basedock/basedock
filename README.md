# BaseDock

BaseDock is a server management system built with .NET 10 and TanStack Start. It provides an admin panel to manage Docker images and Docker Compose configurations.

## Features

- **Docker Management**: Manage images and compose stacks.
- **Admin Panel**: Built with TanStack Start and Shadcn UI.
- **Authentication**: .NET Identity with RBAC.
- **Realtime Updates**: SignalR integration.
- **Architecture**: Clean Architecture with .NET 10 Minimal APIs.
- **Orchestration**: .NET Aspire 13.

## Tech Stack

- **Backend**: .NET 10, Aspire 13, EF Core, PostgreSQL, Redis.
- **Frontend**: TanStack Start, Shadcn UI, Tailwind CSS.

## Getting Started

### Prerequisites

- .NET 10 SDK
- Docker Desktop
- Node.js & pnpm

### Running the Project

1.  Clone the repository.
2.  Run `dotnet run --project src/BaseDock.AppHost`.
